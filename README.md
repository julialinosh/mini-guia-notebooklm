# Mini guia organização financeira - notebooklm
Guia de Organização Financeira Pessoal

 Contexto e Objetivo

Este projeto tem como objetivo estudar e compreender a importância da organização financeira pessoal, utilizando Inteligência Artificial como ferramenta de apoio ao aprendizado.

O tema foi escolhido devido à relevância no dia a dia, já que o controle financeiro é essencial para evitar dívidas, planejar o futuro e alcançar estabilidade econômica.

Os principais objetivos deste estudo são:
- Entender como organizar receitas e despesas
- Aprender técnicas de planejamento financeiro
- Desenvolver hábitos financeiros saudáveis
